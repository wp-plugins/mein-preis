== Description ==
MeinPreis erlaubt dem dir den Preisverlauf eines Produkts auf deiner Seite einzubinden. 
Dazu kann beim erstellen oder bearbeiten einfach "[meinpreis:<amazon asin>]" eingefügt werden.

Ein Beispiel wäre:
[meinpreis:B004Q3QSWQ]
